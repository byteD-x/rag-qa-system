# 文档中心

本目录是 `RAG-QA 2.0` 的正式文档入口，围绕当前双内核架构维护。

## 快速导航

| 文档 | 用途 |
| --- | --- |
| [API 说明](API_SPECIFICATION.md) | 网关、小说服务、企业库服务的接口概览 |
| [开发脚本与本地工作流](dev-scripts.md) | 启动、停止、构建、日志和回归检查 |
| [运行手册](runbook.md) | 常见故障、排障顺序、恢复建议 |
| [Demo 数据集说明](demo-dataset/README.md) | 样例文档与评测数据的使用方式 |

## 当前文档边界

以下内容不视为正式产品说明文档：
- `docs/呢喃诗章(咸鱼飞行家).txt`：小说模式的大文本样本
- `docs/demo-dataset/**/*`：样例输入与评测数据
- `docs/reports/*`：阶段性验证输出

## 维护约定

- 接口、环境变量、启动方式变更时，必须同步更新本目录文档
- 文档以当前默认运行路径为准，不再描述旧三服务架构
- 新增文档优先回答三个问题：做什么、怎么用、怎么验证
