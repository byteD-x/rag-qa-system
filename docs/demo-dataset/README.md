# Demo 数据集说明

本目录存放本地演示、联调和评测使用的样例数据，不属于产品运行时代码。

## 目录结构

| 路径 | 用途 |
| --- | --- |
| `doc_1.txt` ~ `doc_15.txt` | 小型样例文档，适合上传链路演示 |
| `evaluation/` | 基础评测问题集 |
| `adversarial/` | 对抗与异常输入样例 |

## 使用建议

- 小说线路优先使用根目录提供的长文本样本做大文件验证
- 企业库线路可直接使用 `doc_*.txt` 做上传、检索和问答演示
- `evaluation/` 适合做回归验证
- `adversarial/` 适合做拒答或异常处理验证

## 边界

- 这些文件只用于开发、演示和评测
- 不应混入真实客户数据
- 对外发布仓库时可以保留，但不要加入敏感材料
